# Code of Conduct

This project follows a simple rule: **treat other contributors the way you'd want to be treated when you're wrong about something technical in public.**

Practically, that means:

- Focus criticism on the code, not the person
- When someone's code is wrong, the goal is to fix it, not to make them feel bad
- Assume good intent; ask clarifying questions before jumping to conclusions
- Disagreements about technical direction are fine, even expected — keep them grounded in tradeoffs, not ideology
- No harassment, slurs, personal attacks, threats, or deliberate hostility

## Scope

Applies everywhere the project interacts: GitHub issues, PRs, discussions, Discord/Slack channels, Twitter replies mentioning Axon, and any in-person events.

## Enforcement

Violations can be reported privately to `conduct@axon.dev`. Reports are confidential. The maintainers review every report and respond within 72 hours.

Possible outcomes:
- A private conversation
- A public correction
- Temporary suspension from project spaces
- Permanent ban

We don't publish details of individual reports, but we do publish aggregate transparency stats annually.

## Attribution
