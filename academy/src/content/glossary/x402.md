# x402

**x402** é um protocolo aberto que revive o status HTTP **402 Payment Required** — um código reservado desde 1997 mas nunca implementado em larga escala — e o combina com pagamentos em stablecoin (USDC) na Base L2.

## Como funciona (simplificado)

1. Cliente (ex: um agente de IA) faz request pra uma API
2. API responde `402 Payment Required` com headers descrevendo preço e endereço de pagamento
3. Cliente envia transação on-chain pagando o valor
4. Cliente refaz o request com prova de pagamento
5. API verifica on-chain e libera a resposta

Tudo automático, sem humano no meio, sem cartão de crédito, sem assinatura mensal.

## Por que importa

Agentes de IA autônomos não podem usar cartão de crédito (fraude, aprovação humana, faturamento mensal). Eles precisam de pagamento **programático, atomic, micro**. x402 entrega exatamente isso.

## Quem usa

Nexus Inovation é um dos primeiros gateways adotando x402 nativamente. Coinbase publicou a spec em [x402.org](https://x402.org) e mantém o padrão aberto.
