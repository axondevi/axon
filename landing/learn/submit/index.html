Envie seu tutorial

A Academy cresce com ajuda da comunidade. Se você construiu algo interessante com Nexus Inovation — tutorial, recipe, integração, case de uso — queremos publicar.

O que aceitamos

  • Tutoriais: passo-a-passo com código rodável (Python, Node, Go)
  • Recipes: snippet curto (10-30 linhas) resolvendo um problema
  • Guias: artigos conceituais sobre agentes, APIs, arquitetura
  • Traduções: PT-BR ↔ EN de conteúdo existente

Como enviar

  1. Fork do repo: github.com/axondevi/axon
  2. Crie seu arquivo em academy/src/content/tutorials/nome-do-tutorial.mdx (ou guides/, recipes/)
  3. Siga o template em academy/README.md
  4. Abra um Pull Request

Recompensa

  • Byline com seu nome + link pro seu site
  • $20 em créditos Nexus Inovation USDC por PR aceito
  • Compartilhamento no Twitter @axondevia
  • Destaque em "Agent of the Week" se for projeto original

Dúvidas ou propostas?

Abre uma issue no GitHub descrevendo o tema — a gente responde se faz sentido antes de você escrever tudo.

Ou manda DM pra @axondevia.
